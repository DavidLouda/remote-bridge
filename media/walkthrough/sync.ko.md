> **베타 기능** — 동기화 신뢰성은 VS Code Settings Sync에 따라 달라집니다. 다른 장치에 연결이 표시되지 않으면 *지금 연결 동기화 (베타)*를 수동으로 실행하세요.

**장치 간 동기화 작동 방식:**

1. 모든 장치에서 마스터 비밀번호 활성화 (어디서나 **동일한 비밀번호** 사용)
2. 모든 장치에서 VS Code Settings Sync에 로그인
3. 설정에서 `remoteBridge.security.syncConnections` 활성화

연결은 항목별 타임스탬프를 사용하여 자동으로 병합됩니다 — 데이터는 절대 손실되지 않습니다. 삭제는 30일 동안 추적되므로 한 장치에서 삭제된 연결은 다른 장치에서도 삭제된 상태로 유지됩니다.

**참고:** 한 장치에서 마스터 비밀번호를 변경하면 Remote Bridge가 다른 장치에서 새 비밀번호로 잠금 해제를 요청합니다.
