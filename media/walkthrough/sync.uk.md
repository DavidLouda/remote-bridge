> **Бета-функція** — надійність синхронізації залежить від VS Code Settings Sync. Якщо з'єднання не з'являються на іншому пристрої, запустіть *Синхронізувати з'єднання зараз (Бета)* вручну.

**Як працює синхронізація між пристроями:**

1. Увімкніть головний пароль на всіх пристроях (використовуйте **той самий пароль** скрізь)
2. Увійдіть у VS Code Settings Sync на всіх пристроях
3. Увімкніть `remoteBridge.security.syncConnections` у Налаштуваннях

З'єднання об'єднуються автоматично за допомогою часових міток — жодні дані не втрачаються. Видалення відстежуються протягом 30 днів, тому з'єднання, видалене на одному пристрої, залишається видаленим на інших.

**Примітка:** Якщо ви змінюєте головний пароль на одному пристрої, Remote Bridge запропонує розблокувати новим паролем на інших.
