> **ベータ機能** — 同期の信頼性は VS Code Settings Sync に依存します。別のデバイスに接続が表示されない場合は、*接続を今すぐ同期（ベータ）* を手動で実行してください。

**デバイス間同期の仕組み：**

1. すべてのデバイスでマスターパスワードを有効にする（どこでも **同じパスワード** を使用）
2. すべてのデバイスで VS Code Settings Sync にサインインする
3. 設定で `remoteBridge.security.syncConnections` を有効にする

接続はアイテムごとのタイムスタンプを使用して自動的にマージされます — データが失われることはありません。削除は 30 日間追跡されるため、1 つのデバイスで削除された接続は他のデバイスでも削除されたままになります。

**注意：** あるデバイスでマスターパスワードを変更すると、Remote Bridge は他のデバイスで新しいパスワードでのロック解除を求めます。
