**What you get with master password encryption:**

- AES-256-GCM encryption of the entire connection store
- Automatic encrypted backups — 7 daily + 4 weekly, stored locally, never synced
- Ability to sync connections (including passwords) across devices via VS Code Settings Sync
- Restore from backup at any time via *Remote Bridge: Restore from Backup*

**Tip:** The password must be at least 8 characters. Store it safely — it cannot be recovered.
