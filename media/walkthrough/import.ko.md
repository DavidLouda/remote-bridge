**지원 가져오기 소스:**

| 소스 | 가져오는 내용 |
|---|---|
| SSH Config (`~/.ssh/config`) | 호스트, 포트, 사용자, ID 파일, 에이전트 |
| WinSCP (`WinSCP.ini`) | SSH, SFTP, FTP, FTPS — 암호화된 비밀번호 포함 |
| SSH FS (VS Code 확장) | 구성된 모든 원격 서버 |
| FileZilla (`sitemanager.xml`) | FTP, SFTP, FTPS — Base64 비밀번호 자동 디코딩 |
| PuTTY (레지스트리 / `~/.putty`) | SSH 세션 — 키 경로 유지 |
| Total Commander (`wcx_ftp.ini`) | FTP, FTPS — 비밀번호 역난독화 |

**팁:** *Remote Bridge: 연결 내보내기*를 통해 나중에 연결을 내보낼 수도 있습니다 — JSON 형식(재가져오기 가능) 또는 SSH Config 형식.
