**Ce que vous obtenez avec le chiffrement par mot de passe maître :**

- Chiffrement AES-256-GCM de l'ensemble du magasin de connexions
- Sauvegardes chiffrées automatiques — 7 quotidiennes + 4 hebdomadaires, stockées localement, jamais synchronisées
- Possibilité de synchroniser les connexions (y compris les mots de passe) entre appareils via VS Code Settings Sync
- Restauration depuis une sauvegarde à tout moment via *Remote Bridge : Restaurer depuis une sauvegarde*

**Conseil :** Le mot de passe doit comporter au moins 8 caractères. Conservez-le en lieu sûr — il ne peut pas être récupéré.
