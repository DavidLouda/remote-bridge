> **Beta 功能** — 同步可靠性依赖于 VS Code Settings Sync。如果连接未出现在其他设备上，请手动运行*立即同步连接 (Beta)*。

**跨设备同步的工作原理：**

1. 在所有设备上启用主密码（每台设备使用**相同的密码**）
2. 在所有设备上登录 VS Code Settings Sync
3. 在设置中启用 `remoteBridge.security.syncConnections`

连接使用按项目时间戳自动合并 — 不会丢失任何数据。删除操作将被追踪 30 天，因此在一台设备上删除的连接在其他设备上也会保持删除状态。

**注意：** 如果您在一台设备上更改主密码，Remote Bridge 将在其他设备上提示您使用新密码解锁。
