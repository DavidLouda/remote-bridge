**Was Sie mit Master-Passwort-Verschlüsselung erhalten:**

- AES-256-GCM-Verschlüsselung des gesamten Verbindungsspeichers
- Automatische verschlüsselte Sicherungen — 7 täglich + 4 wöchentlich, lokal gespeichert, niemals synchronisiert
- Möglichkeit, Verbindungen (einschließlich Passwörter) über VS Code Settings Sync geräteübergreifend zu synchronisieren
- Wiederherstellung aus Backup jederzeit über *Remote Bridge: Aus Backup wiederherstellen*

**Tipp:** Das Passwort muss mindestens 8 Zeichen lang sein. Bewahren Sie es sicher auf – es kann nicht wiederhergestellt werden.
