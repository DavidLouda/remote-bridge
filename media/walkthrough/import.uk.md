**Підтримувані джерела імпорту:**

| Джерело | Що імпортується |
|---|---|
| SSH Config (`~/.ssh/config`) | Хост, порт, користувач, файл ідентифікації, агент |
| WinSCP (`WinSCP.ini`) | SSH, SFTP, FTP, FTPS — включно із зашифрованими паролями |
| SSH FS (розширення VS Code) | Всі налаштовані віддалені сервери |
| FileZilla (`sitemanager.xml`) | FTP, SFTP, FTPS — паролі Base64 декодуються автоматично |
| PuTTY (реєстр / `~/.putty`) | SSH-сеанси — шляхи до ключів збережені |
| Total Commander (`wcx_ftp.ini`) | FTP, FTPS — паролі деобфусковані |

**Порада:** Ви також можете експортувати з'єднання пізніше через *Remote Bridge: Експортувати з'єднання* — у форматі JSON (можна повторно імпортувати) або у форматі SSH Config.
